# Java
The classic "rock, paper, scissors- shoot!" game, implemented in the [Java](https://www.java.com/en/) language
## Prerequisites
* Java ~ Install it from the java website: https://www.java.com/en/download/
## Run
```
$ cd java
$ java index.java
```
