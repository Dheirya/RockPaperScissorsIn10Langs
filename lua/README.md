# Lua
The classic "rock, paper, scissors- shoot!" game, implemented in the [Lua](https://www.lua.org) language
## Prerequisites
* Lua ~ Install it from the lua website: https://www.lua.org/download.html
## Run
```
$ cd lua
$ lua index.lua
```
