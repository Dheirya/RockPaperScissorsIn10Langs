# Bash
The classic "rock, paper, scissors- shoot!" game, implemented in the Bash shell
## Prerequisites
* Bash ~ It's probably installed, but if not, check out the official bash site: https://www.gnu.org/software/bash/
## Run
```
$ cd bash
$ chmod +x ./index.sh 
$ ./index.sh
```
