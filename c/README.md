# C
The classic "rock, paper, scissors- shoot!" game, implemented in the C language using the [Clang](https://clang.llvm.org) compiler
## Prerequisites
* Clang ~ Install it from the clang website: https://releases.llvm.org/download.html
## Run
```
$ cd c
$ clang index.c
$ ./a.out
```
