# Ruby
The classic "rock, paper, scissors- shoot!" game, implemented in the [Ruby](https://www.ruby-lang.org/en/) language
## Prerequisites
* Ruby ~ Install it from the ruby website: https://www.ruby-lang.org/en/downloads/
## Run
```
$ cd ruby
$ ruby index.rb
```
