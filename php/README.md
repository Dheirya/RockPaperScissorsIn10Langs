# PHP
The classic "rock, paper, scissors- shoot!" game, implemented in the [PHP](https://www.php.net) language
## Prerequisites
* PHP ~ Install it from the php website: https://www.php.net/downloads
## Run
```
$ cd php
$ php -S 127.0.0.1:8000
```
