# Perl
The classic "rock, paper, scissors- shoot!" game, implemented in the [Perl](https://www.perl.org) language
## Prerequisites
* Perl ~ Install it from the perl website: https://www.perl.org/get.html
## Run
```
$ cd perl
$ perl index.pl
```
